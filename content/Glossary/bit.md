# A Bit Is the Smallest Unit of Digital Information Representing One Binary Choice

A bit (binary digit) is the atomic unit of information in digital systems, holding exactly one of two possible values — 0 or 1 — which map to physical voltage levels LOW and HIGH in hardware.

## Explanation

The word "bit" was coined by Claude Shannon in 1948. It represents the answer to a single yes/no question: on or off, true or false, charged or not charged. Every more complex data type — bytes, integers, floating-point numbers, text, images — is ultimately a structured collection of bits.

In hardware, a bit is realised as a voltage level, a magnetic domain orientation, a charge trapped in a transistor gate, or the presence/absence of a pit on an optical disc. The physical representation varies; the logical meaning (0 or 1) is universal.

Eight bits form one byte. Groupings of 4 bits (a nibble) correspond neatly to one hexadecimal digit, which is why hex is the standard shorthand for binary data.

## Key Points

- Bit = binary digit; values: 0 or 1
- 4 bits = 1 nibble = 1 hex digit
- 8 bits = 1 byte
- n bits can represent 2ⁿ distinct values

## Example

A single bit can encode a coin flip (heads = 1, tails = 0). Two bits can encode four outcomes (00, 01, 10, 11). Eight bits can encode 256 distinct values — enough for all ASCII characters.
