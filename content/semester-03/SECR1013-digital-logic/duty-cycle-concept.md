### IDEEA

Duty cycle defines how long a signal stays in the ON state compared to its total period.

### Details

- Formula:
    - Duty Cycle = (ON time / Total period) × 100%
- Example:
    - T = 1000 ms
    - Duty cycle = 25%
    - ON = 250 ms, OFF = 750 ms
- Interpretation:
    - 50% → balanced ON/OFF
    - High % → mostly ON
    - Low % → mostly OFF
- Used in:
    - PWM (Pulse Width Modulation)
    - Power control systems
